## Java Concurrency Deck

## Setting up...

### Install the marp cli

```bash
npm install -g @marp-team/marp-cli
```

### Serve the Deck

```bash
$ marp -s ./
```

### Serve in Watch Mode

```bash
$ marp -w ./concurrency.md
```

### Export the Deck

```bash
marp concurrency.md -o converted.pdf --allow-local-files
```
